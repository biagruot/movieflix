## Run Locally

Clone the project

```bash
  git clone https://github.com/biagruot/movieflix.git
```

Go to the project directory

```bash
  cd movieflix
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Run e2e tests

```bash
  ng e2e
```
